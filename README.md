# CantierIA

**CantierIA** è un servizio web che usa l'intelligenza artificiale per generare computi metrici estimativi a partire da documenti di capitolato.

Il problema è semplice: compilare un computo metrico a mano richiede ore. Bisogna leggere il capitolato voce per voce, cercare la corrispondenza nel prezziario regionale (tra migliaia di voci scritte in burocratese) e riportare tutto in un foglio strutturato. Un lavoro ripetitivo, lento e soggetto a errori costosi.

CantierIA automatizza questo processo. L'utente carica un documento — PDF, Word o testo libero — e l'IA estrae le lavorazioni, le abbina alle voci del prezziario della regione di riferimento tramite ricerca semantica, e produce un computo strutturato pronto per la revisione. Il risultato si può esportare nei formati più usati nel settore: XPWE, XLSX e PDF.
